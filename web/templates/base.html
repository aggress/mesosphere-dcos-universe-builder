Local Universe Builder



A UI for building a DC/OS local universe. Select your DC/OS version, followed by the packages and build.

Search for a package by typing the first few letters of its name, the box will update automatically.

Use your up and down arrows to select the right package, then hit enter to confirm.

Amend selections using the backspace key, or click on a package then delete.

Build time will vary based on the number of packages and resources allocated by Docker.
